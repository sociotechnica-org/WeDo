## Validation Criteria

- A task with schedule_rules `{ "days": ["MO","TU","TH","FR"] }` appears only on those days
- A task scheduled M-Sa does not appear on Sunday
- Sunday displays an empty board (no tasks materialize) with a gentle visual treatment
- Different persons can have different schedules for different tasks
- The recurrence engine correctly evaluates schedule_rules using RFC 5545 day codes
- "Today" is anchored to EST timezone, not UTC

## Motivation

The recurrence engine is WeDo's key differentiator — the "different bit" that makes it more than a static chore chart. Micah's piano practice appears M/Tu/Th/Fr, his vacuuming appears M-Sa, and neither appears on Sunday. Without this, someone has to manually create tasks every day.
