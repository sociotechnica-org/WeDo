## Validation Criteria

- Dashboard View renders in landscape with one column per person (up to 6)
- Each column shows: streak count, completion ring, person name, task list
- Task rows display checkbox + emoji + task text
- Completion ring fills proportionally as tasks are completed
- Updates from any connected client appear on all other clients within 1 second
- Dashboard functions as an ambient display on iPad in full-screen Safari

## Motivation

The dashboard is the product. WeDo exists to give a family a shared, ambient view of everyone's daily tasks. If the dashboard doesn't render correctly with real-time sync, nothing else matters. This outcome proves the full vertical stack works end-to-end.
