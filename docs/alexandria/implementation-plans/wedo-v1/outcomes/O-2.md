## Validation Criteria

- Tapping a person's column on the dashboard opens their Single List View
- Tapping anywhere on a task row toggles it done/not-done
- Completion ring updates immediately on toggle
- At 100% completion, ring shows a simple warm visual treatment (subtle color shift)
- Toggling a task on one client updates all other connected clients in real time
- Past-day tasks can be toggled for catch-up entry
- Tasks can be deleted via swipe (mobile) or hover-reveal trash icon (desktop)

## Motivation

Task completion is the core daily interaction — the thing every family member does multiple times a day. The toggle must be instant, satisfying, and reflected everywhere. Without this, the daily completion rhythm loop doesn't fire.
