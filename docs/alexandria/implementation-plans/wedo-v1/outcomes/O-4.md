## Validation Criteria

- User types "practice piano Monday Tuesday Thursday Friday" and a task is created with correct schedule_rules and a suggested emoji
- The Anthropic API (Sonnet 4.6) parses natural language into structured task data via tool_use
- The Zod schema validates the API response before writing to D1
- Ambiguous input gets a reasonable best-guess parse (not an error)
- Created task appears immediately on the dashboard for the correct person and days

## Motivation

Natural language entry is one of WeDo's three strategic bets. Describing tasks in plain language ("vacuum Monday through Saturday") instead of fighting a recurrence picker is the experience that makes WeDo feel effortless. This is a Should because the app functions with manually seeded tasks, but it's the primary entry mechanism for ongoing use.
