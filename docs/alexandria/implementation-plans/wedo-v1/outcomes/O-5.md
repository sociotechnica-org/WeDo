## Validation Criteria

- Streak count (consecutive 100% days) displays above each person's name on the dashboard
- Completing all tasks for a day increments the streak the next day
- Missing any task resets the streak to zero
- Skip days (toggled via "SKIP TODAY" on the date bar) preserve the streak — the day doesn't count
- Skip days dim all tasks for that day visually
- Sunday is automatically excluded from streak calculation (never counts)
- Editing past-day tasks triggers retroactive streak recalculation from the edited day forward

## Motivation

Streaks are the engagement amplifier — the secondary loop that makes the daily completion rhythm sticky over time. Skip days are the pressure valve that prevents streaks from becoming punitive. Together they create the "we're on a roll" family motivation that WeDo is designed for.
